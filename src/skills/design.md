# Guía de Estilo Visual: WorldTeams

## 1. Paleta de Colores (Sistema Hexadecimal)

### Colores Principales y de Fondo
- **Night:** `#131313` (Negro base para texto principal y fondos oscuros).
- **Dark Green:** `#142D2B` (Verde corporativo profundo para identidad).
- **Snow:** `#FFF7F3` (Blanco roto/Crema para fondos claros y legibilidad).

### Colores de Acento y Categorías
- **Mindaro:** `#D2F176` (Verde ácido/Lima - Acento principal para destacar futuro e innovación).
- **Hollywood Cerise:** `#F900A8` (Rosa vibrante para elementos dinámicos).
- **Pale Dogwood:** `#D4B5AA` (Nude/Arena para equilibrio visual).
- **Persian Green:** `#58958C` (Verde medio para elementos secundarios).
- **Dark Purple:** `#3E0D2A` (Púrpura profundo para contrastes elegantes).
- **Lavender Pink:** `#FFBDF4` (Rosa suave para detalles o fondos sutiles).

## 23. Tipografía

- **Títulos (Headings):** `Space Grotesk`, sans-serif.
  - *Uso:* H1 a H4. Carácter geométrico y tecnológico.
- **Cuerpo de texto (Body):** `Manrope`, sans-serif.
  - *Uso:* Párrafos y UI general. Alta legibilidad.
- **Acentos y Citas:** `Nyght Serif`, serif.
  - *Uso:* Destacar frases, conceptos narrativos o elementos decorativos ("Human touch").
- **Logotipo (Referencia):** `Helvetica Neue` (Bold para "World", Light para "Teams").

## 3. Especificación de Tema Oscuro (Dark Mode)

Reglas de aplicación para el agente de IA:
- **Fondo Primario:** Utilizar **Night** (`#131313`).
- **Texto Principal:** Utilizar **Snow** (`#FFF7F3`).
- **Contraste de Secciones:** Usar **Dark Green** (`#142D2B`) para tarjetas o contenedores sobre el fondo negro.
- **Acentos:** Priorizar **Mindaro** (`#D2F176`) para botones o llamadas a la acción (CTA) sobre fondo oscuro, ya que ofrece el mayor contraste.
- **Bordes/Divisores:** Usar **Persian Green** (`#58958C`) con opacidad reducida.

## 4. Instrucciones para la IA
1. **Código CSS:** Utilizar variables raíz basadas en los nombres oficiales (ej. `--color-mindaro`).
2. **Jerarquía Visual:** Combinar `Space Grotesk` (Títulos) con `Manrope` (Cuerpo). Usar `Nyght Serif` solo para dar énfasis o un toque sofisticado.
3. **Elementos Gráficos:** Si se generan imágenes o componentes, incluir formas hexagonales (referencia al Isotipo "Jewel") y tramas de líneas que representen "conectividad global".
4. **Tono de Voz:** Profesional, inspirador y directo. Evitar un lenguaje excesivamente frío; mantener el enfoque en las personas.
